Rishvi Inventory — Inventory Operations

Rishvi Inventory enables seamless synchronization of inventory data between Odoo and Linnworks. It automates warehouse imports, category management, stock updates, and bulk inventory synchronization.

Inventory Synchronization Features

1. Warehouse Synchronization

  • Imports all Linnworks warehouses into Odoo
  • Creates or updates warehouse records automatically
  • Maintains accurate warehouse mapping

2. Product Category Sync

  • Imports categories and subcategories
  • Preserves parent–child hierarchy
  • Keeps product classification consistent

3. Add Inventory by Linn SKU

Manually update inventory for a specific product using its Linn SKU. Ideal for quick corrections without full sync.

4. Bulk Inventory Sync

  • Multi-page inventory import
  • Batch-based synchronization
  • Perfect for first-time setup or full refresh

POS with Linnworks Integration

Shipping services, discounts, pricelists, and categories are imported directly from Linnworks via API. POS orders are automatically synced back to Linnworks with order numbers.

POS Sales Flow

  1. Open POS Session
  2. Select or Add Customer
  3. Add Products and Apply Discounts
  4. Select Shipping Service
  5. Validate Order
  6. Automatic Sync to Linnworks
