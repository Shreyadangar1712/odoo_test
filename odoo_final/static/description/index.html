Inventory & Multi-Warehouse Synchronization

Rishvi Inventory enables structured and automated inventory operations between Rishvi (Odoo) and Linnworks.

It supports warehouse imports, product categories, SKU-based updates, and bulk inventory synchronization, ensuring both systems remain fully aligned.

Warehouse Import

Sync all warehouses configured in Linnworks into Rishvi Odoo. Warehouses are created or updated automatically with correct internal location mapping.

Bulk Inventory Sync

Perform large-scale inventory imports using multi-page batch processing. Supports controlled syncing of hundreds of products per operation.

SKU-Level Adjustments

Update inventory for individual products using Linnworks SKU. Ideal for quick corrections without running a full inventory sync.

Error-Safe Sync Logic

Inventory comparisons and validations prevent data mismatches and overselling, maintaining accurate stock across systems.

On-Hand Quantity

Represents the real-time physical inventory available in the warehouse. This quantity reflects actual stock present and ready for sale or internal use, synchronized continuously between Rishvi and Linnworks.

POS System Features

Rishvi POS is designed for fast billing, accurate stock control, and seamless backend synchronization.

It simplifies daily sales operations while ensuring inventory and accounting data remain consistent.

Product Selection - Quickly browse, search, and add products to the order list.
Payment Processing - Supports cash, card, and digital payment methods.
Discount Application - Apply discounts per product or on the full order.
Live Inventory Sync - Stock quantities are automatically updated when orders are validated.
Receipt Generation - Generate printable or digital receipts instantly.
Offline Mode - Continue billing without internet and auto-sync once connected.
Barcode Scanning - Speed up billing using barcode scanners.
User Access Control - Role-based permissions for cashiers and managers.
Accounting Integration - Sales, taxes, and payments are posted automatically to accounting journals.
Session End Summary - Detailed session report with sales totals, payments, and cash difference.
